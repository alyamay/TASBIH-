# TASBIH+

**TASBIH+** adalah aplikasi mobile yang menyediakan alat dzikir digital dan kumpulan doa harian dengan tampilan sederhana, nyaman, dan mudah digunakan. Aplikasi ini ditujukan untuk membantu pengguna berdzikir dan membaca doa secara terstruktur sekaligus menyimpan riwayat ibadah yang telah dilakukan.

## Tujuan Aplikasi
- Menyediakan tasbih digital sebagai pendamping dzikir kapan saja.
- Menyediakan daftar doa harian dengan UI sederhana dan ramah pengguna.
- Mendukung tampilan Light Mode dan Dark Mode.
- Menyediakan akses multiuser melalui fitur Login & Register.
- Menyimpan riwayat terakhir dzikir (Last Count) dan doa terakhir yang dibaca (Last Read).

## Fitur Utama

| Fitur | Penjelasan |
|------|------------|
| **Login & Register** | Mendukung multiuser (akan dikembangkan). |
| **Home Page** | Menampilkan salam, Last Read, Last Tasbih, dan Islamic Quotes. |
| **Doa List Page** | Menampilkan daftar doa harian. Data masih menggunakan dummy JSON (akan dihubungkan API). |

| **Tasbih Digital** | Counter dzikir dengan tampilan lingkaran progres. Progress disimpan dengan SharedPreferences. |
| **Favorite Page** | Menyimpan doa sebagai favorit (CRUD). |
| **Settings Page** | Mengubah mode tema & mengedit nama pengguna (akan dikembangkan). |
| **Bottom Navigation**| Untuk navigasi ke setiap halaman. |
